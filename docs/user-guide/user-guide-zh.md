# Skills Vis — 使用者指南

Skills Vis 是一款互動式的 AI agent 技能庫瀏覽器。它讓你以圖形方式視覺化你的技能、直接在瀏覽器中編輯技能、即時監控 agent 使用了哪些技能，以及從遠端儲存庫匯入技能。

本指南將帶你完成所有入門步驟，並充分運用每一項功能。

---

## 系統需求

- Python 3.10 或以上版本
- 現代瀏覽器（Chrome、Firefox、Safari、Edge）

---

## 安裝

從 PyPI 安裝 Skills Vis：

```bash
pip install skills-vis
```

---

## 啟動應用程式

在終端機中啟動伺服器：

```bash
skills-vis
```

也可以用 Python 模組方式啟動：

```bash
python -m skills_vis
```

然後在瀏覽器中開啟：

```
http://127.0.0.1:8001
```

---

## 載入技能資料夾

應用程式開啟後，頁面頂端會看到一個路徑輸入欄位。

1. 貼上你的技能資料夾路徑（例如 `~/.claude/plugins/my-skills`）。
2. 點擊 **Load（載入）**。

圖形將會顯示，以節點樹的方式呈現你的技能資料夾。

> **小提示：** 你啟動 `skills-vis` 時所在的目錄，會自動出現在技能抽屜中作為快速載入項目，不需要重新輸入路徑。

## 瀏覽技能

![圖形視圖](../../static/screenshots/01_graph.png)

圖形以從左到右的樹狀結構顯示你的技能資料夾：

- **展開／收合資料夾** — 點擊資料夾節點上的箭頭，或直接點擊資料夾名稱。
- **選取技能** — 點擊任一檔案節點，即可在右側檢視面板中開啟它。
- **搜尋** — 按下 `⌘K`（Mac）或 `Ctrl+K`（Windows/Linux）開啟搜尋列。輸入文字以依名稱篩選節點。符合的節點會被誆選標註。你仍可以手動收合被標註的資料夾。

---

## 開啟並編輯技能

![檔案檢視器](../../static/screenshots/03_editor.png)

點擊圖形中的任一檔案節點，即可在檔案檢視器中開啟它：

- **渲染預覽** — 技能的 Markdown 內容渲染，右側顯示即時目錄。
- **編輯模式** — 點擊 **Edit（編輯）** 按鈕（鉛筆圖示）切換至程式碼編輯器。完成修改後點擊 **Save（儲存）** 。

---

## 管理工作區

![技能抽屜](../../static/screenshots/02_drawer.png)

點擊左上角的抽屜圖示，開啟**技能抽屜**。在這裡你可以：

- **切換工作區** — 點擊任一已儲存的工作區項目，即可立即將該資料夾載入到圖形中。抽屜保持開啟，方便你持續切換。
- **儲存目前資料夾** — 使用 **Save current root（儲存目前根目錄）** 快速操作，將目前載入的資料夾加入書籤。
- **新增資料夾** — 使用 **New skill folder（新增技能資料夾）** 輸入新路徑並儲存。
- **編輯或刪除**已儲存的項目，使用各項目旁的圖示操作。
- **入門技能庫** — 瀏覽精選的入門技能，並直接加入你的工作區。

## 即時監控 Agent

![錄製工作階段](../../static/screenshots/04_monitor.png)

Skills Vis 可以在 AI agent 運作時，即時標註被呼叫的Skill。

### 開始錄製工作階段

1. 點擊工具列上的 **⏺ Record（錄製）** 開始新的工作階段。
2. 將 `skills-monitor/SKILL.md` 技能載入到你的Agent工作階段中（將它加入Agent讀取的技能資料夾）。
3. 正常使用你的Agent。Agent每次呼叫技能時，對應節點會在圖形上短暫發出**綠色**光芒。
4. 點擊工具列上的 **Recording…（錄製中…）** 停止工作階段。

### 重播過去的工作階段

過去的工作階段會自動儲存。開啟**技能抽屜**，在 **Recordings（錄製）** 區段找到對應的工作階段，點擊即可在圖形上重播技能呼叫的順序。

## 從 GitHub / GitLab 匯入技能

![匯入技能](../../static/screenshots/05_import.png)

你可以從任何 GitHub、GitLab 儲存庫，或本機路徑匯入技能。

1. 點擊 **Browse…（瀏覽…）** 按鈕，或工具列上的匯入圖示。
2. 貼上網址或路徑，例如：
   - `https://github.com/mattpocock/skills`
   - `https://gitlab.com/example/skills-repo`
   - `/Users/yourname/projects/shared-skills`
3. Skills Vis 會掃描儲存庫中的 `SKILL.md` 檔案並列出清單。
4. 點擊任一技能可**預覽**其內容。
5. 勾選你想要的技能，然後點擊 **Copy selected（複製選取項目）** 將它們匯入到你目前的工作區資料夾。

> **衝突偵測：** 如果工作區中已存在同名技能，Skills Vis 會在覆蓋前向你發出警告。

## 小技巧與鍵盤快捷鍵

| 快捷鍵          | 功能                       |
| --------------- | -------------------------- |
| `⌘K` / `Ctrl+K` | 開啟搜尋                   |
| 點擊資料夾箭頭  | 展開／收合資料夾           |
| 點擊檔案節點    | 在檢視器中開啟技能         |
| 點擊工作區項目  | 載入工作區（抽屜保持開啟） |

- **啟動目錄：** 你執行 `skills-vis` 時所在的資料夾，會固定顯示在技能抽屜頂端，即使你尚未儲存它也一樣。
- **Python 模組啟動：** `python -m skills_vis` 與 `skills-vis` 完全相同 — 當 CLI 入口點不在你的 PATH 中時特別有用。
